# Grades Application

#### Grades application uses [GradesAPI](https://github.com/hasangurbuzz/grades-api) for storing and finding student grades to help teachers.

## Techs

- ReactJS
- TypeScript
- OpenAPI Generator
- Redux with Redux Toolkit
- React Router
- React Bootstrap
- Axios
